Υλοποίηση στο rbt folder σε C της δομής δεδομένων Red Black Tree με κλειδί έναν ακέραιο (int) και τις παρακάτω ιδιότητες:

1. Εφαρμογή καλών πρακτικών υλοποίησης ενοτήτων.

2. Υποστήριξη ενός οποιοδήποτε τύπο κλειδιού, π.χ. String, Struct (αφαίρεση πράξεων "=" και συγκρίσεων "<", ">", "==", κλπ).

3. Οργάνωση Ενοτήτων με Ολική Απόκρυψη.

4. Υποστήριξη genericity.
