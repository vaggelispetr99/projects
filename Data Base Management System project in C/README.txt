Η υλοποίηση των δύο εργασιών του Εργασία 2022-2023 pdf, βρίσκεται στους δύο
φακέλους με όνομα DB_Project_1 και DB_Project_2.

Η λειτουργικότητα των δομών, και στις δύο εργασίες, 
επιδεικνύεται στα main αρχεία κώδικα, τα οποία βρίσκονται
εντός του φακέλου examples. Τα στατιστικά δεδομένα,
εμφανίζονται και αυτά στα main αρχεία.

Στην πρώτη εργασία, εμφανίζονται τα στατιστικά
δεδομένα που αφορούν στο primary hash table, ενώ
στην δεύτερη εμφανίζονται τα statistics και για
το primary και για το secondary hash table.

Για όλες τις δομές, εισάγεται ένα πλήθος από entries
και αναζητούνται κάποιες τυχαίες εγγραφές (στο HP
και στο HT με βάση τα IDs και για το SHT, με βάση
τα ονόματα).

To build και το τρέξιμο των αρχείων, ακολουθεί τον
τρόπο που περιγράφηκε και στο readme της εκφώνησης
της εργασίας, οπότε μπορεί επί τόπου ο χρήστης να
ελέγξει τις υλοποιήσεις μας.

Η εργασία έτρεξε σε GNU C Compiler, από Linux.
Παρακαλείσθε να μην χρησιμοποιήσετε άλλο compiler,
διότι δεν εγγυόμαστε ότι θα τρέχει ομαλά (σίγουρα
αποφύγετε τον MinGW, από Windows, διότι κάνει
διαφορετική default αρχικοποίηση των μεταβλητών).
